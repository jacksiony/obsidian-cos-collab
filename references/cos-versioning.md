# 开启 COS 版本控制

> 版本控制是这套系统的保险丝，必须开启。

## 为什么需要版本控制？

AI 协作中不可避免会：
- 发布错误内容
- 生成重复、无用的笔记
- 误删或覆盖重要文件

COS 版本控制可以：
- 查看文件历史版本
- 回滚到任意时间点
- 永久保留被删除的对象

## 开启步骤

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 进入 **对象存储 COS**
3. 选择你的存储桶
4. 左侧菜单：**基础配置** → **容错容灾管理**
5. 找到 **版本控制**，点击 **开启**
6. 确认（可能需要等待几分钟生效）

## 验证是否开启

```bash
# 上传一个测试文件
coscmd upload test.txt /test-version.txt

# 再上传一次（覆盖）
coscmd upload test.txt /test-version.txt

# 查看版本列表（COS 控制台）
# 应该看到两个版本
```

在控制台文件详情页，可以看到「历史版本」标签。

## 回滚操作

1. 进入存储桶，找到目标文件
2. 点击文件 →「历史版本」
3. 选择要恢复的版本
4. 点击「恢复到当前版本」

## 注意事项

- 版本控制会**占用额外存储空间**（每个版本都单独存储）
- 建议定期清理不需要的历史版本
- 不要关闭版本控制（除非你确定不再需要回滚）
